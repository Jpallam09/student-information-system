The Student Records Information System is a web-based application designed to help schools, teachers, and administrators manage student data efficiently. It provides a centralized platform to store, view, and update student information, including personal details, academic records, and class schedules.

This system aims to simplify administrative tasks, reduce manual paperwork, and improve data accuracy and accessibility.

Key Features

Student Management – Add, update, view, and delete student records.

Teacher Management – Maintain teacher profiles and assign them to classes.

Section & Schedule Management – Organize students by sections and view class schedules.

Role-Based Access – Different access for administrators, teachers, and students.

Search & Filters – Quickly find students or teachers using dynamic filters.

Secure Login System – Authentication for teachers and students.

Technologies Used

Frontend: HTML, CSS, JavaScript

Backend: PHP

Database: MySQL

Server Environment: XAMPP (Apache + MySQL)
